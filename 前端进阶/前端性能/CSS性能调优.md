### 提升 CSS 选择器性能
　　CSS 选择器对性能的影响源于 **浏览器匹配选择器和文档元素时所消耗的时间**，所以优化选择器的原则是应尽量避免使用消耗更多匹配时间的选择器。**注意：CSS 选择器是从右到左进行规则匹配。**
  * 避免使用通用选择器
  * 避免使用标签或 class 选择器限制 id 选择器
  * 避免使用标签限制 class 选择器
  * 避免使用多层标签选择器。使用 class 选择器替换，减少 CSS 查找
